# Week6 Homework 
## Synopsis 
1. Write a function bool isPrime( unsigned int
   i ) that determines whether a number is
   prime. Use this function in a program that
   prints all prime factors of a positive
   integer input by the user. The maximum
   number we enter is 65535.
2. Write a recursive function power( base,
   exponent ) that, when invoked, returns
   base^exponent
3. Write a recursive function gcd that returns
   the greatest common divisor of x and y
4. Write a function lcm that returns the
   lowest common multiple of x and y
5. Write a function that displays a solid
   square of asterisks(*) whose side is
   specified in integer parameter side.
6. Please complete the recursive function f to
   calculate the nth Fibonacci number.
